# Nursery Management System

A comprehensive nursery management system built with React.js frontend and Node.js backend.

## Features

### For Parents
- Track child attendance and activities
- View daily reports and photos
- Make online payments
- Receive real-time notifications
- Message teachers and staff

### For Employees/Teachers
- Record daily attendance
- Log activities and observations
- Manage classroom activities
- Report medical alerts
- Communicate with parents

### For Administrators
- Manage users (parents, employees, children)
- Generate invoices and reports
- Monitor system capacity
- View analytics and dashboards

## Tech Stack

### Frontend
- React.js 18
- React Router for navigation
- Context API for state management
- Axios for API calls
- CSS3 for styling

### Backend
- Node.js with Express.js
- MongoDB with Mongoose ODM
- JWT for authentication
- bcryptjs for password hashing
- PDFKit for invoice generation
- Nodemailer for email notifications
